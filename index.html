WinLIRC.NET

WinLIRC.NET is a software framework developed using Microsoft .NET framework v4.0 over the open source WinLIRC program. It provides a .NET wrapper over the existing WinLIRC program to be integrated with any .NET applications for Infra-Red signal communication.
